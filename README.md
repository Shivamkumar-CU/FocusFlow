# 🎯 FocusFlow  

FocusFlow ek lightweight productivity timer hai jo tumhe apne work/study sessions ko manage karne me help karta hai. Ye project simple UI ke sath Pomodoro-style focus sessions create karta hai taaki tumhara concentration aur efficiency dono improve ho.  

---

## 🚀 Live Demo  
👉 [FocusFlow on GitHub Pages](https://shivamkumar-cu.github.io/FocusFlow/)  

---

## ✨ Features  
- ⏳ Countdown Timer (minutes & seconds support)  
- ▶️ Start, ⏸ Pause aur ⏹ Reset controls  
- 🎨 Responsive & Clean UI with CSS styling  
- 💻 Works directly in browser (no extra setup needed)  

---

## 🛠️ Tech Stack  
- **HTML5** – Structure  
- **CSS3** – Styling & Layout  
- **JavaScript (ES6)** – Timer Logic
